# GW-1 零碳辐射制冷涂料 · 商业计划书（苹果风格单页 HTML）

面向 **A 轮股权融资（1.5 亿元）** 的对外商业计划书，采用苹果产品页视觉风格的单页滚动式网站。所有财务、概率、估值数据均由可复现的 Python 模型重算校验，数据来源于《GW-1 可行性研究报告 v2.0（OEM 轻资产模式升级版）》。

**在线访问**：https://bistuwangqiyuan.github.io/gw-1-radiative-cooling-business-plan/

---

## 一、目录结构

```
tuliao/
├── index.html                      # 商业计划书主页面（单页滚动）
├── assets/
│   ├── css/style.css               # 苹果风格样式
│   ├── js/
│   │   ├── main.js                 # 交互、滚动动效、ECharts 图表
│   │   └── echarts.min.js          # 本地内置图表库（离线可用）
│   ├── data/
│   │   ├── model_output.json       # 模型输出（数据底座）
│   │   └── model_data.js           # 同上，挂载到 window.MODEL_DATA 供前端加载
│   └── img/                        # 高端配图（hero/tech/applications/network/vision）
├── models/
│   ├── financial_model.py          # 可复现财务 + 蒙特卡洛 + 敏感性模型
│   └── requirements.txt            # 依赖（numpy）
└── GW-1…可行性研究报告_v2.0…docx     # 原始可研报告（数据来源）
```

## 二、如何查看

- **在线版（推荐对外分享）**：https://bistuwangqiyuan.github.io/gw-1-radiative-cooling-business-plan/
- **方式一（本地）**：直接双击 `index.html` 用浏览器打开。图表库与数据均为本地文件，无需联网。
- **方式二（本地服务）**：在项目根目录启动本地服务后访问 `http://localhost:8765/index.html`：

```powershell
cd D:\project\cursor\bp\tuliao
python -m http.server 8765
```

> 建议使用基于 Chromium 的浏览器（Chrome/Edge）或 Safari，以获得最佳的毛玻璃、滚动动效与字体效果。

## 三、如何复现全部数据

```powershell
cd D:\project\cursor\bp\tuliao
pip install -r models\requirements.txt
$env:PYTHONIOENCODING="utf-8"
python models\financial_model.py
```

运行后会：
1. 在控制台打印「重算值 vs 报告披露值」对账表；
2. 重新生成 `assets/data/model_output.json` 与 `assets/data/model_data.js`。

蒙特卡洛使用固定随机种子 `SEED = 20260607`，因此任何人重跑都会得到完全一致的结果。

| 参数 | 取值 |

| 所得税率 | 15%（高新技术企业） |

| 估值倍数 | PE 25× / PS 6× |

## 五、数据对账表（重算值 vs 报告披露值）

### 确定性指标（完全复现）

| 指标 | 重算值 | 报告值 | 差异 |
|---|---|---|---|
| NPV（亿元） | 21.06 | 21.06 | 0.00 |
| IRR（%） | 99.7 | 99.7 | 0.00 |
| 终值 TV（亿元） | 32.32 | 32.32 | 0.00 |
| 静态回收期（年） | 4 | 4 | 0 |
| 5 年累计净利（亿元） | 5.68 | 5.68 | 0.00 |
| Y5 估值-PE 法（亿元） | 44.41 | 44.41 | 0.00 |
| Y5 估值-PS 法（亿元） | 54.00 | 54.00 | 0.00 |
| Y5 估值-综合（亿元） | 49.21 | 49.21 | 0.00 |

> NPV 口径说明：NPV = 折现自由现金流 + 折现终值（合计约 22.06 亿）− t0 初始股权 1.0 亿 = 21.06 亿，与报告一致。
> Y5 综合估值口径：取 PE 法（44.41）与 PS 法（54.00）均值 = 49.21 亿，与报告一致。

### 蒙特卡洛指标（50,000 次，误差均在约 1 个百分点内）

| 指标 | 重算值 | 报告值 | 差异 |
|---|---|---|---|
| P(NPV>0) 成功率（%） | 92.36 | 91.87 | +0.49 |
| P(强成功)（%） | 56.47 | 58.33 | −1.86 |
| P(卓越)（%） | 3.68 | 4.33 | −0.65 |
| P(失败)（%） | 7.64 | 8.13 | −0.49 |
| P(破产)（%） | 4.38 | 5.51 | −1.13 |
| NPV_P5（亿元） | −0.79 | −1.25 | +0.46 |
| NPV_P95（亿元） | 28.32 | 29.19 | −0.87 |

> 蒙特卡洛采用报告 9.2 节给出的 11 变量分布与 R8 推广延迟权重（50%/30%/20%）；
> 由于原报告未公开其蒙特卡洛源码，本模型为独立重建，结果与报告高度吻合（差异 ≤ ~1pp），属正常仿真随机性与建模口径差异。

### 需特别说明的差异项（实事求是）

| 指标 | 重算值 | 报告值 | 说明 |
|---|---|---|---|
| ROIC（%） | 58.2 | 63.1 | 报告未披露 ROIC 计算口径。本模型口径为「末年 NOPAT ÷ 投入资本（累计 CAPEX + 末年净营运资本）」。页面与对比图采用本模型重算值 58.2%，以保证可复现、可检验。 |
